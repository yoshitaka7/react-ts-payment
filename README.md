## Getting Started
`git clone`

↓

ルートディクトリに.envファイル作成、stripeのテスト環境用の公開鍵と秘密鍵を記述

```
REACT_APP_STRIPE_PUBLIC_KEY=公開鍵
STRIPE_SECRET_KEY=秘密鍵
```

↓

`npm run start`
